Filtering

Temporarily hide data that doesn't meet your criteria, making it easier to focus on specific information without deleting anything.

Quick Steps:

  1. Select the column or data range
  2. Go to the Data tab in the ribbon
  3. Click the Filter button (funnel icon)
  4. Use dropdown arrows to select values to display

Watch how it works:

When to use:

  • Analyzing specific subsets of data
  • Finding data that meets certain criteria
  • Focusing on relevant information
  • Removing outliers temporarily

Pro Tip:

Use multiple filters across different columns to narrow your data even further. Click the "Clear" button in the Data tab to remove all filters at once.

Sorting

Arrange data in a specific order (alphabetical, numerical, chronological) to organize and analyze patterns more easily.

Quick Steps:

  1. Select the data range including headers
  2. Go to the Data tab
  3. Click Sort button
  4. Choose column and sort order
  5. Add levels for multi-column sorting

Watch how it works:

When to use:

  • Finding highest/lowest values quickly
  • Grouping related data together
  • Creating alphabetical lists
  • Preparing data for reports

Pro Tip:

Always select all related data before sorting to keep rows together (CTRL + A selects everything for you). For quick sorts, select a single cell in the column and use the A-Z/Z-A buttons in the Data tab.

Deleting Columns

Remove unnecessary columns to streamline your data and focus only on relevant information for your analysis.

Quick Steps:

  1. Select column(s) by clicking the column letter(s)
  2. Right-click and select "Delete"
  3. Or use Home tab > Delete > Delete Sheet Columns

Watch how it works:

To delete multiple non-adjacent columns:

→ Hold Ctrl and click columns by their top letter (A, C, E, etc)

→ Right-click > Delete

When to use:

  • Removing irrelevant data
  • Cleaning up imported datasets
  • Preparing data for specific analysis
  • Creating streamlined reports

Pro Tip:

Always make a backup of your workbook before deleting columns. Consider hiding columns (right-click > Hide) instead of deleting if you might need the data later.

Organize different datasets or analysis in separate sheets within the same workbook for better organization and clarity.

Quick Steps:

  1. Click "+" tab at bottom of sheet tabs
  2. To rename: Double-click tab OR right-click > Rename
  3. Type new name and press Enter

Watch how it works:

When to use:

  • Creating backup sheets for your assessment
  • Separating different data categories

  • Building dashboards with source data hidden

Pro Tip:

Right-click a sheet tab for more options like color-coding tabs, moving/copying sheets, or grouping sheets to edit multiple sheets at once. Use meaningful sheet names that clearly describe the content.

MINIMUM / MAXIMUM Function

Find the smallest (MIN) or largest (MAX) value in a range of cells to quickly identify extreme values in your data set.

Function Syntax:

=MIN(number1, [number2], ...)

=MAX(number1, [number2], ...)

OR

=MIN(range)

=MAX(range)

Examples:

=MIN(B2:B15)

// Returns the smallest value in cells B2 through B15


=MAX(C5:F5)

// Returns the largest value in row 5 from columns C to F


Watch how it works:

When to use:

  • Finding highest or lowest values in data sets
  • Identifying outliers or exceptional performances
  • Setting threshold values for conditional formatting
  • Creating dynamic reports that highlight extremes
  • Setting boundaries for valid data entry

IF Function

Perform logical tests and return different values based on whether a condition is true or false, helping you make your data more dynamic and responsive.

Function Syntax:

=IF(logical_test, value_if_true, value_if_false)

Examples:

=IF(A1>10, "High", "Low")

// Returns "High" if the value in cell A1 is greater than 10, otherwise returns "Low"


=IF(B5="Yes", 100, 0)

// Returns 100 if cell B5 contains "Yes", otherwise returns 0


=IF(C2>=70, "Pass", "Fail")

// Returns "Pass" if the value in C2 is 70 or higher, otherwise returns "Fail"

Watch how it works:

When to use:

  • Creating conditional calculations
  • Assigning grades or categories based on values
  • Building dynamic reports with different outputs
  • Error checking or data validation
  • Creating custom alerts or highlights

Conditional Formatting

Automatically change cell appearance based on values or conditions, making it easier to visualize trends, patterns, and exceptions in your data.

Quick Steps:

  1. Select the cells to format
  2. Go to Home tab > Conditional Formatting
  3. Choose a rule type (Highlight Cells Rules, Top/Bottom Rules, Data Bars, Color Scales, Icon Sets)
  4. Set your condition criteria and formatting
  5. Click OK to apply

Watch how it works:

Common rule types:

→ Highlight Cells Rules: Format cells greater than, less than, between, etc.

→ Top/Bottom Rules: Format top/bottom 10 items or top/bottom 10%

→ Data Bars: Add colored bars to visualize relative values

→ Color Scales: Apply color gradients from min to max values

→ Icon Sets: Add visual indicators (arrows, traffic lights, etc.)

When to use:

  • Highlighting values above/below thresholds
  • Visualizing performance against targets
  • Creating dashboards with visual indicators
  • Identifying trends in large data sets
  • Drawing attention to exceptional values

AVERAGE Function

Calculate the arithmetic mean (average) of a set of values to find the central tendency of your data.

Function Syntax:

=AVERAGE(number1, [number2], ...)

OR

=AVERAGE(range)

Examples:

=AVERAGE(A1:A10)

// Averages all values in cells A1 through A10


=AVERAGE(B5,C5,D5)

// Averages the values in those three specific cells

Watch how it works:

When to use:

  • Finding typical values in your data
  • Calculating student grade averages
  • Determining average sales, costs, or time
  • Creating baseline metrics for comparison

Pro Tip:

AVERAGE ignores text and empty cells but counts zeros. For more specialized averages, look at MEDIAN (middle value) or MODE (most common value) functions.

SUM Function

Add up a range of numbers quickly to calculate totals in your data.

Function Syntax:

=SUM(number1, [number2], ...)

OR

=SUM(range1, [range2], ...)

Examples:

=SUM(B2:B10)

// Adds all values in cells B2 through B10


=SUM(A1:A5,C1:C5)

// Adds values from two separate ranges

Watch how it works:

When to use:

  • Calculating column or row totals
  • Summing sales, expenses, or quantities
  • Building financial models
  • Creating running totals

Pro Tip:

Use the AutoSum button (Σ) in the Home tab for quick sums. For complex conditions, try SUMIF or SUMIFS functions to sum only values that meet specific criteria.

COUNTIF Function

Count cells that meet specific criteria, allowing you to summarize how many items match certain conditions.

Function Syntax:

=COUNTIF(range, criteria)

Examples:

=COUNTIF(A1:A20, "Completed")

// Counts how many cells contain "Completed"


=COUNTIF(B5:B100, ">50")

// Counts how many values are greater than 50

Watch how it works:

When to use:

  • Counting occurrences of specific values
  • Tracking frequency of statuses or categories
  • Analyzing data distribution
  • Counting items above/below thresholds

Creating Charts

Transform numerical data into visual representations to identify trends, patterns, and relationships more easily.

Quick Steps:

  1. Select the data range including headers
  2. Go to the Insert tab
  3. Click the chart type you want (column, line, pie, etc.)
  4. Use Chart Design and Format tabs to customize
  5. Right-click chart title to edit text

Watch how it works:

To create a quarterly sales chart:

→ Select data with quarters and sales figures

→ Insert tab > Column Chart

→ Right-click title > Edit Text > Type "Quarterly Sales"

→ Chart Design tab > Add Chart Element > Axis Titles

When to use different chart types:

  • Column/Bar: Compare values across categories
  • Line: Show trends over time
  • Pie: Show parts of a whole (percentages)
  • Scatter: Show relationships between variables

Pro Tip:

Click the plus button when you have the chart selected and tick the 'Axis Titles' button to add them to your chart. You can also double click to edit what they say.

Find and Replace

Search for specific data and replace it with new values, making it easy to correct or standardize information across your spreadsheet.

Quick Steps:

  1. Press Ctrl+H to open Find & Replace
  2. Enter the text to find
  3. Enter the replacement text
  4. Click "Replace All" or "Replace" (to confirm each change)

Watch how it works:

To standardize formatting of state abbreviations:

→ Press Ctrl+H

→ Find what: "California"

→ Replace with: "CA"

→ Click "Replace All"

When to use:

  • Standardizing inconsistent terminology
  • Correcting common typos or errors
  • Updating product codes or IDs
  • Removing unwanted characters

Pro Tip:

If you leave the "Find" section blank, it will fill any empty cells with whatever is in your "Replace" section for you.
